COMPASS v2.3.5 — ROWLIE TUCKED LEFT
=====================================

This version:
- Restores your approved Rowlie head and compass image
- Makes Rowlie smaller
- Tucks Rowlie into a narrow left column
- Gives the COMPASS cards the remaining page width
- Keeps the larger yellow Rowlie introduction only once
- Keeps the Drive authorization/upload fix
- Keeps all student, teacher, Coordinator, goals, grades, portfolio,
  Showcase, passport, and project features

INSTALL
-------
1. Open Extensions > Apps Script.
2. Replace all of Index.html with:
   Compass_Index_v2_3_5_to_Copy.txt
3. Replace all of Code.gs with:
   Compass_Code_v2_3_5_to_Copy.txt
4. Save both files.

DRIVE AUTHORIZATION
-------------------
If not already completed:
1. Select authorizeCompassDrive from the function dropdown.
2. Click Run.
3. Approve the requested Drive permissions.

DEPLOY
------
1. Deploy > Manage deployments.
2. Click Edit.
3. Confirm Execute as: Me.
4. Choose New version.
5. Description: Compass v2.3.5 Rowlie Left
6. Deploy.
